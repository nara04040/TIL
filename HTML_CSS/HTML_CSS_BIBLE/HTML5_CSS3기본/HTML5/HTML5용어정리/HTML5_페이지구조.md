# 페이지 구조

```html
<!DOCTYPE html>
<html>
    <head>
        <title>HTML5 Basic Page</title>
    </head>
    <body>

    </body>
</html>
```

> 코드 해석
- !DOCTYPE html : 웹 브라우저가 현제 HTML5문서임을 인식하게 한다. 반드시 문서의 가장 첫 번째 줄에 있어야한다.

- html 태그에 lang속성 추가 가능

- head태그 내부에 작성 가능한 태그
  - meta : 페이지에 추가 정보를 전달
  - title : 페이지의 제목
  - script : 스크립트를 추가한다.
  - link : 다른 파일을 추가한다.
  - style : 스타일시트를 추가한다.
  - base : 기본경로를 지정한다.
